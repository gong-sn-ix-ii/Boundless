





คำสั่งนั้นใช้ไม่ได้เพราะว่าเครื่องคอมพิวเตอร์ของคุณยังไม่มีโปรแกรม Google Cloud SDK ติดตั้งอยู่ครับ ซึ่ง gcloud เป็นคำสั่งหลักที่มากับ SDK นี้

## วิธีติดตั้ง Google Cloud SDK (สำหรับ Windows)
ทำตามขั้นตอนต่อไปนี้เพื่อติดตั้งครับ:

ดาวน์โหลดตัวติดตั้ง:

ไปที่หน้าดาวน์โหลดของ Google Cloud SDK: https://cloud.google.com/sdk/docs/install

คลิกที่ปุ่มดาวน์โหลดสำหรับ Windows (มักจะเป็นไฟล์ ...-windows-x86_64-interactive.exe)

รันตัวติดตั้ง:

เปิดไฟล์ .exe ที่ดาวน์โหลดมา

ทำตามขั้นตอนในโปรแกรมติดตั้ง (ส่วนใหญ่สามารถกด "Next" ไปได้เรื่อยๆ)

สำคัญ: ในหน้าสุดท้ายของการติดตั้ง ให้แน่ใจว่าคุณได้ติ๊กถูกที่ช่อง "Run 'gcloud init' to initialize the gcloud CLI"

## ขั้นตอนหลังติดตั้ง: ตั้งค่าโปรเจกต์ (gcloud init)
หลังจากติดตั้งเสร็จ โปรแกรมจะเปิดหน้าต่าง Command Prompt สีดำขึ้นมาโดยอัตโนมัติเพื่อรันคำสั่ง gcloud init ให้คุณทำตามขั้นตอนต่อไปนี้:

ล็อกอิน:

ระบบจะถามว่า You must log in to continue. ให้พิมพ์ Y แล้วกด Enter

เบราว์เซอร์ของคุณจะเปิดขึ้นมา ให้ ล็อกอินด้วยบัญชี Google เดียวกับที่คุณใช้สร้างโปรเจกต์ Firebase

เมื่อล็อกอินสำเร็จ ให้กลับไปที่หน้าต่าง Command Prompt

เลือกโปรเจกต์:

ระบบจะแสดงรายชื่อโปรเจกต์ทั้งหมดของคุณ ให้คุณมองหาโปรเจกต์ boundless-1304c

พิมพ์ หมายเลข ที่อยู่ข้างหน้าชื่อโปรเจกต์นั้น แล้วกด Enter

ตั้งค่า Region (ไม่บังคับ):

ระบบอาจจะถามให้ตั้งค่า default region คุณสามารถกด Enter ผ่านไปก่อนได้

เมื่อขั้นตอนทั้งหมดนี้เสร็จสิ้น Google Cloud SDK ก็จะพร้อมใช้งานครับ

## เมื่อติดตั้งเสร็จแล้ว
ให้คุณ เปิดหน้าต่าง Command Prompt หรือ PowerShell ขึ้นมาใหม่ (เพื่อรีเฟรชระบบ) แล้วลองรันคำสั่งเดิมอีกครั้ง:

Bash

gcloud run services add-iam-policy-binding agora --member="allUsers" --role="roles/run.invoker" --region="us-central1"
ครั้งนี้คำสั่งควรจะทำงานได้สำเร็จ และ Permission ของฟังก์ชัน agora ของคุณก็จะถูกตั้งค่าอย่างถูกต้องครับ
